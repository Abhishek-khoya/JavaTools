# JavaTools
**Setter / Getter Generator** 

It is a handy tool to generate setter and
getter methods of class. Used Java Reﬂection API.

**POJO Copier **

It is a tool to make a shallow copy of Plain Old Java
Object (POJO), Used Java Reﬂection API.

**Keyboard Class **

It is a replica of Java Scanner class, Like Scanner
class of java, user can get input from input stream in the speciﬁed
format using appropriate method of this Keyboard class.
